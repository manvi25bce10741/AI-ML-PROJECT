PROBLEM STATEMENT: Students in the Computer Science branch often get confused about where to study from because there are too many platforms available online. They usually end up switching between different websites and waste a lot of time deciding instead of actually studying. Different platforms are better for different purposes like concepts, coding practice, or quick revision, but students are not always sure which one to choose. So, there is a need for a simple tool that can help students decide the most suitable platform based on what they need at that moment. This project aims to build a program that takes basic inputs from the user and recommends platforms using a simple scoring approach.
SCOPE: The system is mainly designed for Computer Science students and beginners who need guidance in choosing study resources. It can also be useful for students preparing for exams or placements who want quick suggestions instead of searching manually. The project is limited to basic recommendation logic using predefined rules and is mainly for learning and demonstration purposes. It is not intended to replace detailed academic planning or personalized mentoring.
TARGET AUDIENCE: This system is designed for students who are unsure about which platform to use for studying subjects like DSA, DBMS, OS, CN, and Programming. It is helpful for beginners who are still exploring different resources and need some direction. It can also be useful for students who have limited time and want quick suggestions based on their goals. Apart from that, it is suitable for learners who are studying Python, since the project demonstrates simple concepts like functions, conditions, and basic decision-making.
FEATURES: The system includes the following main features:
>User Input Collection
Takes inputs such as subject, goal, learning preference, and available time from the user.
>Recommendation Logic
Uses a simple scoring method to decide which platforms are more suitable.
>Scoring System
Assigns scores to different platforms based on the inputs provided by the user.
>Ranking of Platforms
Sorts the platforms according to their scores.
>Top Suggestions
Displays the top 2 recommended platforms instead of just one.
>Score Display
Shows how much score each platform received so the user can understand the decision.
>Simple Output
Displays the results in a clean and easy-to-read format.
